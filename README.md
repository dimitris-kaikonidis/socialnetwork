### Social network for Musicians

#### Check out on heroku https://bandm8s.herokuapp.com/

#### Functionalities:

* Create account
* Upload Picture
* Crop Picture
* Reset Password and sending reset code to email using AWS SES
* Likes & Comments
* Recommendations based on user skills
* Friends
* Chat

#### Tech Stack:

* React
* Redux
* Node.js
* Express
* PostgreSQL
* Redis
* Socket.io
* AWS
